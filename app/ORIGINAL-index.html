About: Factornine

Factornine Limited is the company which represents Russell Wenban a London FED developer.

I have been programmer for 15 years and have developed a variety of projects for leading e-learning companies, design agencies and start-ups.

Some of the well known brands I've produced for include: BBC Jam, Espresso Education, Ford, Virgin, Nectar, Videotel, Anchor Butter and McDonald's.

Skills: AS to JS

I am a proficient programmer successfully making the transition from the Classical world of ActionScript to the contemporary world of functional JavaScript and Angular.js.

My skills include proficiency in the following: AS3, JavaScript, Angular.js, iOS Development, Ionicframework, HTML5 and CSS3, SASS, Twitter Bootstrap, UNIX, Gulp, Grunt and ANT and Gulp.

I'm recently updated this website to be a `Yeoman web app` and have added a portfolio written in Angular using TweenMax animation and created this dynamic `skills matrix` using d3 in Angular.js.

I'm currently working on creating e-learning apps using Ionic for the Apple App Store.

Magic Square Game

I've continued my interest in creating e-learning games by creating a new game for my portfolio.

The Magic Square Puzzle is a classic logic game, where each row, column, and diagonal adds up to the same number. This number is the Magic Number.

My version of this puzzle is written in Angular and is an open source project hosted on Github. See: Magic Squares.
