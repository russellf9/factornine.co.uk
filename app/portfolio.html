Factornine - Angular.js - Proof of Concepts

Pontoon

'Pontoon' is a multiplayer game written in Angular.js using Twitter Bootstrap and implements the Shephertz API. I've used a Finite State Machine to marshall the apps current state and pass JSON messages between users to synchronise the model between the players. The game is still very much in progress and has been a great project for me to implement my skills in a new sphere.

Videotel International - E-learning - 2010 - 2012

Videotel the worlds leading provider of marine training and education. My role include maintenance of the existing AS3 framework and development of new features.

I was principally responsible for the development of the following 'widgets':

  • 'Case Study' - a 'Skeuomorphic' animated flash asset with video and text.
  • 'Zoomable Part' - a widget which allows the user to zoom into a section of an image and then select further information.
  • 'Media Explorer' - a widget which allows the user to select hotspots which trigger the opening of a window with further media and text.
  • 'Prompts' - a selection of modal windows which present the user with 'characters', text or media.
  • 'Media Launcher' - a view with links to various 'prompts'.
  • 'Image Carousel' - a widget which is a carousel for various media.
  • 'Document Viewer' - a widget which displays documents in SWF format. The widget can be configured to display the asset full width or height and controls allow the user to zoom in and out.
  • 'Video Player' - a bespoke video component which includes controls to provide multilingual support using SRT files triggered by FLV cue points.
  • Test/Quiz - Extensive work was carried out to improving Videotel's 'Test Engine'. The student can be taught by learning through hints, the test results are recordable and this data can be assigned to the clients database. Types of question include: multiple choice and various drag and drop questions. Each is configurable and in 'tutorial' mode can provide feedback to the user.
